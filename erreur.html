Vous utilisez une ancienne version de Microsoft Internet Explorer. L'application "Ordres de mission & Frais de mission" n'est pas compatible avec les versions de Microsoft Internet Explorer inférieures à 9.

Vous pouvez utiliser Mozilla Firefox ou Google Chrome (navigateurs internet gratuits).

Vous pouvez télécharger un navigateur gratuit compatible :

  • Mozilla Firefox : https://download.mozilla.org/?product=firefox-36.0-SSL&os=win&lang=fr
  • Google Chrome : https://www.google.fr/chrome/browser/desktop/index.html
Après installation de Firefox ou de Chrome, vous aurez accès à l'application.
